Fun Fact:


Computer Programming Language Diversity.

There are many countries that speak more than one language. According to VistaWide, Papua New Guinea is the country with the highest number of languages (820), accounting for 11.86% of the world’s languages. Indonesia comes as a close second with 742 languages, and Nigeria with more than 500. Now imagine there’s a country where the people only speak in computer languages. If that happened, it would, at the very least, knock Nigeria off the third place! In fact, Wikipedia currently lists over 700 programming languages. However, this list doesn’t feature some languages, like Markup languages (an example is HTML.) There are also other lists, such as FOLDOC, that includes more than 1,000 languages.
